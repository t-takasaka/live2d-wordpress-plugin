# Live2D WordPress Plugin

WordPress で Live2D を動かすプラグインです。

管理画面が付いているのでファイルパスやモデルの位置・サイズの変更が手軽にできます。

また、マウスカーソルの方向へ顔や視線を向ける機能と、モデルをクリックしたときにモーションを切り替える機能が付いています。
